# FlashMind

FlashMind คือเกมฝึกความจำและการสังเกต ผู้เล่นจะต้องจดจำสีที่ปรากฏบนหน้าจอ และตอบให้ถูกต้องว่าสีใดแสดงผลในเวลาสั้นที่สุด

ตัวเกมถูกพัฒนาด้วย TypeScript, Vite, HTML และ CSS พร้อมดีไซน์สไตล์ modern minimal

---

## วิธีการเล่น

1. สังเกตสีที่แสดงบนหน้าจอทีละสี
2. แต่ละสีจะปรากฏในระยะเวลาไม่เท่ากัน
3. จำให้ได้ว่าสีไหนแสดงสั้นที่สุด
4. เลือกคำตอบให้ถูกต้องเพื่อผ่านด่าน

หากตอบผิด เกมจะจบทันที

---

## ฟีเจอร์ภายในเกม

- ระบบ Level
- ระบบเพิ่มความยาก
- ระบบ Game Over
- ระบบ Restart
- UI สไตล์ Modern Minimal
- Animation และ Effect ภายในเกม

---

## เทคโนโลยีที่ใช้

- TypeScript
- Vite
- HTML5
- CSS3
