# FlashMind — Memory Challenge

FlashMind คือเกมฝึกความจำและสมาธิ ที่ออกแบบมาเพื่อทดสอบความสามารถในการสังเกตและจดจำรายละเอียดภายในเวลาสั้น ๆ

ผู้เล่นจะต้องจำว่า:

> “สีไหนแสดงบนหน้าจอสั้นที่สุด”

เมื่อเกมเริ่ม สีต่าง ๆ จะถูกแสดงทีละสีด้วยระยะเวลาที่แตกต่างกัน และผู้เล่นต้องเลือกคำตอบที่ถูกต้องหลังจบรอบ

ยิ่ง Level สูงขึ้น:

* จำนวนสีจะมากขึ้น
* เวลาการแสดงจะเร็วขึ้น
* ความยากในการสังเกตจะเพิ่มขึ้น

---

## ฟีเจอร์ภายในเกม

* Countdown ก่อนเริ่มเกม
* ระบบ Level Progression
* ระบบสุ่มสีแบบ Dynamic
* ระบบกำหนดเวลาแสดงสีอย่างเป็นธรรม
* เอฟเฟกต์ Glow และ Smooth Transition
* UI สไตล์ Minimal Futuristic
* ระบบ Replay
* Keyboard / Mouse Interaction

---

## วิธีเล่น

1. สังเกตสีที่แสดงบนหน้าจอ
2. จำว่า “สีไหนแสดงสั้นที่สุด”
3. เลือกคำตอบหลังจบรอบ
4. หากตอบถูก จะเข้าสู่ Level ถัดไป

---

## ระบบความยาก

เกมจะค่อย ๆ เพิ่มความยากตาม Level:

| Level | จำนวนสี |
| ----- | ------- |
| 1 - 3 | 3 สี    |
| 4 - 6 | 4 สี    |
| 7 - 9 | 5 สี    |
| 10+   | 6 สี    |

นอกจากนี้ ระยะเวลาการแสดงสีจะค่อย ๆ ลดลง เพื่อเพิ่มความท้าทายในการจดจำ

---

## จุดเด่นของเกม

* ทุกสีในแต่ละรอบถูกกำหนดเวลาอย่างชัดเจน
* ระบบถูกออกแบบให้ “แฟร์” กับผู้เล่นทุกคน
* ไม่มีการล็อกคำตอบตายตัว
* ใช้ทั้งสมาธิ ความจำระยะสั้น และการสังเกต

## วิธีติดตั้ง

Clone โปรเจกต์:

```bash id="fmd1"
git clone https://github.com/your-username/flashmind.git
```

ติดตั้ง dependencies:

```bash id="fmd2"
npm install
```

เริ่มรันโปรเจกต์:

```bash id="fmd3"
npm run dev
```

Build สำหรับ Production:

```bash id="fmd4"
npm run build
```
